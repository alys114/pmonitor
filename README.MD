# 需求
主机管理：
通过前端页面实现操作后台数据库：实现增删改查
实现主机分组
实现不同的用户可以管理不同的主机
在前端可以对主机信息实现增删改查


# 实现思路
- 使用Django来搭建网站的架构；
- 数据库访问，暂时使用SQLAlchemy；
- 数据库连接的是MySQL；

# 部署环境
1. Python3.5版本以上；
2. 在Linux上运行；
3. 数据库采用MySQL5.5及以上，根据配置建立数据库和用户，并分配权限；
4. 数据初始化脚本：/Db/pm_init.sql
5. Django 2.0
